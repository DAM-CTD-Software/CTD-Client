Usage
=====

There are mainly three things this software can be used for: initiating a CTD
measurement using a Sea-Bird CTD with the Seasave software, configuring
and running a CTD processing procedure, as well as configuring and using Near-
Real-Time data publications. The benefit of doing it through this
interface is the automated metadata addition to data and about the processing
process. Additionally, it allows the programable closure of CTD water bottles
by specification of target depth values.
