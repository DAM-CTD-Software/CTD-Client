Bugs
====

If you encounter any bugs or have other ideas for improvement do not hesitate
to `reach out <mailto:emil.michels@io-warnemuende.de>`_.
If you have an IOW-Git-Account you can also file an `issue <https://git.io-warnemuende.de/CTD-Software/CTD-Client/issues>`_.
If you are searching for the CTD-Clients log-files, they can be found inside
the users directory -> AppData -> Local -> ctdclient -> ctdclient -> Logs
